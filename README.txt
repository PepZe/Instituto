Activate your POP/IMAP from gmail

Settings -> Advanced settings -> POP/IMAP -> State -> Activate IMAP

Add your emails credentials in the Form\view

Install pyqrcode to run the generate QrCode
$ pip install pyqrcode
$ pip install pypng

Add your database information in the settings in var DATABASE
in the same file add your Secret Key

*The current method send email work just for gmail, you can edit the STMTP host and port for another email
